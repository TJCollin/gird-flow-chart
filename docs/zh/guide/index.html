# Grid FLow Chart 是什么

Grid Flow Chart 是一款用于画流程图的插件。核心是通过画网格来确定流程图节点位置，给定连接线起始节点和最终节点，自动确定连接线轨迹。

# 画图步骤

  • 确定流程图所在区块

  • 在上述区块中划分网格

  • 确定流程图每个节点所在网格位置

  • 确定需要连线的起始点
